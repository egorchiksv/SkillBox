4.7 Итоги пройденных тем. Проверьте себя
Описание
Это задание для самостоятельного выполнения, его не нужно сдавать на проверку.

Цель практической работы
Научиться работать с терминалом.

Что нужно сделать
Примените команды и утилиты терминала для работы с файлами и директориями.

Создайте структуру каталогов ~/skillbox/test/ в домашней директории пользователя.
Создайте пустой файл file1 в директории ~/skillbox/.
Запишите вывод утилиты cal в созданный файл file1.
Скопируйте файл в директорию ~/skillbox/test/.
Переименуйте файл file1 в директории ~/skillbox/вfile0.
Перейдите в директорию /etc.
Выведите содержимое каталога /etc в файл ~/skillbox/file2.
Найдите подстроки host в файле file2 и допишите их в конец файла file1. Выведите десять первых и последних строк файла file1.
Вернитесь в домашнюю директорию.
Выведите историю переходов по каталогам.
Найдите в домашней директории пользователя (включая поддиректории) все файлы (искать нужно только файлы), в имени которых есть цифра 2.


Советы и рекомендации
Уделите особое внимание утилите cal. В некоторых версиях Linux эта утилита называется ncal, но её функционал остаётся таким же.

Также если утилиты cal/ncal отсутствуют в ОС, то следует установить их при помощи команды на выбор:

sudo apt-get install cal 
sudo apt-get install ncal
Для вывода утилиты cal в созданный файл file1 выполните команду с перенаправлением данных в файл > или >> . 



Полезные материалы
Путь к файлу в Linux
Перенаправление потока вывода в файл
Поиск вхождений строки
